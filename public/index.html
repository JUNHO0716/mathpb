Mathbee 이용 안내
선생님의 시간을 아끼는 도구가 되겠습니다.
선생님들의 구독료는 디자인·사용자 편의·핵심 기능 개발에 투명하게 재투자됩니다. 현장에서 필요한 기능을 빠르게 반영하고, 업무 시간을 줄여드리는 것을 최우선으로 하겠습니다.

1) 현재 제공 기능

  • 시험지 요청 서비스
    선생님께서 직접 올려주신 파일을 바탕으로,
    해설과 미주가 포함된 완성형 한글(HWP) 시험지를 제작
    👉 추가 비용 없이 구독만으로 이용 가능합니다.
  • 기출문제 다운로드
    다양한 기출을 PDF·HWP로 내려받기
  • 📢 현재 문제은행은 개발 중이라 시험지 위주로 운영 중이며, 구독료를 할인하여 제공하고 있습니다.

2) 문제은행 (제작 중)

단원·유형·난이도 등 원하는 조건으로 문제를 조합해 시험지를 만들 수 있도록 준비 중입니다. AI 기반 유사문항 추천과 개념 추가 기능을 포함하며, 결과물은 모두 한글(HWP) 다운로드를 지원합니다.

3) 내 책장 기능 확장

  • 내가 직접 제작한 시험지 저장·관리
  • 학년·단원·유형별 폴더를 만들어 여러 시험지를 체계적으로 정리
  • 문제은행에서 고른 문제에 개념을 추가해 만드는 맞춤형 교재 관리
  • AI 편집으로 디자인·레이아웃을 원하는 형태로 조정 가능한 교재 제작 전용 공간 제공 예정

Mathbee의 약속

선생님의 구독에 책임 있게 보답하겠습니다. 매달 현장에 꼭 필요한 개선을 반영하고, “선생님의 시간을 아껴주는 파트너”가 되겠습니다.
